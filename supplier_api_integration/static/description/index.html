Installation & Setup

Install the Module

Go to Apps → Search "Supplier API Integration" → Click Install

Configure API Credentials

Go to Inventory > Configuration > Suppliers API > API TME → Enter token and secret

Test Connection

Click "Test Connection" button → Verify credentials are valid

Fetch Categories

Click "Fetch Categories" → Wait for supplier category tree to load

Import Products

Browse categories → Click Preview → Click Import → Crons handle the rest

How It Works

Smart Anti-Duplicate Engine

When importing a product:

1. Search by Barcode (EAN) - Global search → Found? Update existing product → Not found? Continue to step 2 2. Search by SKU - Default code or API ID → Found? Update existing product → Not found? Create new product RESULT: Zero duplicates. Existing products updated, new products created.

Auto-Publishing Logic

Stock > 0 → Product automatically published on website

Stock = 0 → Product automatically unpublished (hidden)

Respects "Continue Selling" override for pre-orders

Mixed Inventory Display

Customers see both warehouse AND supplier stock:

Scenario 1: Warehouse 4 units + Supplier 7 units → "4 in stock (1-2 days)" + "7 available remotely (4-8 days)" Scenario 2: Warehouse 0 + Supplier 7 + Continue Selling enabled → "7 available remotely (4-8 days)" + "Back-order available"

Automatic Cron Jobs

Hourly: Update Stock & Prices

Runs: Every hour

Updates: Stock quantities, cost prices, product names, images, website categories, auto-publish/unpublish flag

Daily: Full Mirror Sync

Runs: Every day at 2 AM

Updates: Complete category tree, all product fields, detects removed products, repairs broken hierarchies
